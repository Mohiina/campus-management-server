## Overview
This repository contains the **server-side API** for the Campus Management System.
The backend is responsible for handling all data operations related to campuses
and students, including database interactions and API request processing.

## Tech Stack
- Node.js
- Express.js
- PostgreSQL
- Sequelize (ORM)

## What This Server Does
- Exposes RESTful API endpoints for campuses and students
- Performs Create, Read, Update, and Delete (CRUD) operations
- Manages relationships between campuses and enrolled students
- Validates and persists data in a PostgreSQL database

## Project Structure
The backend follows a modular structure with separate folders for:
- Database models
- API routes
- Application configuration
